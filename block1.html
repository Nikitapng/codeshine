Блок 1. Редактор кода
Visual Studio Code

Установка Visual Studio Code:

Перейдем на сайт code.visualstudio.com, далее нажимаем на кнопку "Download" и скачиваем установочный файл для вашей операционной системы (Windows, macOS, Linux).

После того как программа скачана, запускаем инсталлятор и следуем инструкциям: принимаем, добавляем все галочки, далее, ждем пока программа установится на наш ПК. Откройте Visual Studio Code, чтобы убедиться, что установка прошла успешно.

Интерфейс программы:

Боковая панель: по умолчанию в левой части редактора расположилась панель действий с 6 элементами

Рабочая область: это основное окно, где вы будете писать и редактировать код. Здесь отображаются открытые файлы и их содержимое. Рабочая область поддерживает множество вкладок, что позволяет работать с несколькими файлами одновременно.

Установка расширений:

Программа запущена, и мы можем начать знакомство с ней. Чтобы облегчить этот процесс, установим русский язык. Для этого нужно нажать на иконку расширений или нажать комбинацию клавиш CTRL + SHIFT + X, а затем в поле поиска ввести "russian". Продолжайте вводить, пока не появится пакет "Russian Language Pack for Visual Studio Code".

Теперь нажимаем на кнопку "Install". После этого программа предложит перезапуститься, чтобы изменить язык. Однако, если вы владеете английским языком, переходить на русский совсем не обязательно.

Затем установим расширение Live Server, которое поможет вам быстро и легко просматривать изменения в вашем веб-проекте. Для установки требуется снова найти в поисковой строке расширение и установить его.

Создание проекта:

Откройте проводник, нажав на иконку или CTRL + SHIFT + E. Нажмите на кнопку "Открыть папку" и создайте новую папку в удобном для вас месте, например, в папке "Документы". Назовите её my-project. Внутри этой папки нажмите на кнопку "Файл" с плюсом и создайте файл index.html. Затем создайте папку css, нажав на кнопку "Папка" с плюсом.
