Блок 3. Стилизация сайта
при помощи CSS

CSS (Cascading Style Sheets) — это язык, который используется для оформления веб-страниц. С его помощью можно изменять внешний вид HTML-документов, например, менять цвета, шрифты, размеры и расположение элементов.

CSS-правила

CSS состоит из правил, которые говорят браузеру, как должны выглядеть различные части веб-страницы. Каждое правило состоит из двух основных частей: селектора и блока свойств.

Селекторы

Селектор — это часть CSS-правила, которая указывает, к каким HTML-элементам будут применены стили. Селектор находится в начале правила, до фигурных скобок {}.

Свойства и значения

Каждое CSS-правило содержит блок свойств, который определяет, как будет выглядеть элемент. Свойства записываются внутри фигурных скобок { } и состоят из пары "свойство: значение". Например:

Здесь color и font-size — это свойства, а red и 14px — их значения.

Распространенные свойства

1. color — устанавливает цвет текста.

  • Можно задать цвет с помощью названия (например, blue), HEX-кода (например, #0066FF) или RGB-значения (например, rgb(0, 17, 255))

2. font-size — задаёт размер шрифта.

  • Можно указать размер в пикселях (px), процентах (%), em или других единицах

3. font-family — определяет тип шрифта.

  • Указывается название шрифта или семейство шрифтов. Можно перечислить несколько шрифтов через запятую.

4. background-color — устанавливает цвет фона элемента.

  • Аналогично свойству color, можно использовать названия цветов, HEX-коды или RGB-значения

5. background — комплексное свойство, которое позволяет задать фон элемента, включая цвет, изображение, повторение и положение.

  • Можно указать несколько значений через пробел

6. margin — задаёт внешние отступы элемента.

  • Можно указать одно значение для всех сторон, два значения (верх-низ и лево-право) или четыре значения (верх - top, право - right, низ - bottom, лево - left)

7. padding — задаёт внутренние отступы элемента.

  • Аналогично margin, можно указать одно, два или четыре значения

8. text-align — выравнивает текст внутри элемента.

  • Возможные значения выравнивая: left(по левому краю), right(по правому краю), center(по центру), justify(по ширине)

9. border-radius — Задаёт радиус скругления углов элемента

  • Можно указать одно значение для всех углов или разные значения для каждого угла.

10. box-shadow — добавляет тень к элементу

  • Задаётся горизонтальное и вертикальное смещение, размытие и цвет тени.

11. width — устанавливает ширину элемента

  • Можно задать в пикселях (px), процентах (%) или других единицах.

12. height — устанавливает высоту элемента

  • Аналогично width, можно задать в пикселях или процентах.

13. position — определяет метод позиционирования элемента

  • Возможные значения: static(позволяет элементу находиться в обычном его состоянии), relative(позиционируется в соответствии с нормальным потоком документа), absolute(удаляется из обычного потока документа, и для элемента в макете страницы не выделяется дополнительное пространство), fixed(фиксируется на одном месте), sticky(фиксируется на одном месте, но передвигается за скроллом).

14. bottom — Задаёт смещение элемента снизу, если используется с position: absolute или position: fixed

  • Указывается значение смещения.
