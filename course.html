Курс 1. Основы верстки на HTML и CSS

Блок 1. Редактор кода Visual Studio Code

Изучим основные особенности программы, фишки, плагины и возможности. Рассмотрим базовую верстку на основе самого популярного редактора кода среди разработчиков сайтов.

Блок 2. Базовая верстка на HTML

Рассмотрим ключевые особенности языка гипертекстовой разметки. Изучим базовую структуру HTML, а так же подробно рассмотрим, из чего состоят элементы этого языка

Блок 3. Стилизация сайта при помощии CSS

Научимся стилизовать HTML - разметку сайта через CSS, подключим CSS в HTML - файл, рассмотрим привязку тегов HTML к CSS - тегам, а так же изучим базовую анимацию и функции таблицы стилей

Блок 3.1. CSS. Флексы

Изучим верстку сайтов с помощью Flex-Box, освоим основные термины, а так же освоим на примерах основные теги и их значения, а так же как они себя ведут на реальных страницах.
